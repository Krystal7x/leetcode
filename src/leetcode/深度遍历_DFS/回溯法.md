1、回溯法框架（其实就是深度遍历搜索DFS，重点！！！）
所有的回溯就像是在遍历一颗多叉树，前中后序遍历就可以了


#####牛逼######
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return

    for 选择 in 选择列表:
        # 做选择
        将该选择从选择列表移除
        路径.add(选择)
        backtrack(路径, 选择列表)
        # 撤销选择
        路径.remove(选择)
        将该选择再加入选择列表
